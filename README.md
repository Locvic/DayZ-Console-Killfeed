# DayZ-Console-Killfeed
Basic DayZ Killfeed for PS4/PS5/Xbox Nitrado Servers

![](https://i.ibb.co/5sjNKQx/image.png)

# Requirements
- Server for hosting
- Python 3.8+
- A brain

# Installation
Install requirements:
`pip3 install -r requirements.txt`

Edit configuration:
`nano config.py` or open in notepad

Run the bot:
`python3 main.py`
